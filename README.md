# 🏆 Copa do Mundo FIFA — Site Informativo
 
Projeto web desenvolvido como **Checkpoint 2** da disciplina de Front-End, apresentando informações sobre a Copa do Mundo FIFA de forma visual e interativa.
 
---
 
## 📋 Descrição
 
Site estático em HTML/CSS que reúne conteúdo histórico, recordes e curiosidades sobre a Copa do Mundo FIFA. O layout conta com navegação fixa, seções temáticas, linha do tempo e cards de curiosidades.
 
---
 
## 📁 Estrutura do Projeto
 
```
projeto/
├── index.html
├── css/
│   └── style.css
└── imagens/
    ├── taca.webp
    └── bola.jpg
```
 
---
 
## 🗂️ Seções
 
| Seção | Descrição |
|---|---|
| **Título / Hero** | Apresentação principal com o troféu da Copa |
| **Sobre** | Descrição geral do torneio e sua relevância global |
| **História** | Narrativa histórica desde 1930 até os dias atuais |
| **Linha do Tempo** | Marcos históricos de cada edição da Copa |
| **Recordes** | Maiores recordistas e estatísticas do torneio |
| **Curiosidades** | Cards com fatos curiosos sobre o torneio |
 
---
 
## 🛠️ Tecnologias Utilizadas
 
- **HTML5** — estrutura semântica das páginas
- **CSS3** — estilização e layout (via `./css/style.css`)
- **Google Fonts** — fontes *Bebas Neue* e *Roboto*
- **Font Awesome 6.5** — ícones decorativos e de navegação
---
 
## 🚀 Como Executar
 
1. Clone ou baixe o repositório.
2. Certifique-se de que a pasta `css/` contém o arquivo `style.css` e que a pasta `imagens/` contém `taca.webp` e `bola.jpg`.
3. Abra o arquivo `index.html` diretamente no navegador — nenhuma instalação adicional é necessária.
